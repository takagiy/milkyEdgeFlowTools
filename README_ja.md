# milkyEdgeFlowTools

[English](README.md) | 日本語

Blender 5.0+ 向けのエッジフロー編集ツール集。現在は 1 つのオペレータを提供します。

**交差するフローをリラックス (Relax Crossing Flows)** — 選択したエッジループを近似曲線上でリサンプルし、そこに交差するエッジループをリラックスさせます。選択ループの形状は曲線によって維持され、頂点の分布だけが変化します。各頂点は、交差するフローが自然に到達したい位置へスライドします。

## 仕組み

- 選択エッジはチェーンに分解されます（開ループ・部分選択・閉ループに対応。分岐のある選択は警告してスキップ）。
- 各チェーンは元の全頂点を通るセントリペタル Catmull-Rom スプラインでフィットされます。
- 各頂点について、交差エッジループの進入方向を曲線上に外挿してリラックス位置を求めます。
- 交差エッジがオブジェクトの形状を定義している頂点（メッシュ境界、シャープな折れ、Sharp マーク付きエッジ）は固定され、その影響は距離に応じて滑らかに減衰します。

## インストール

1. [Releases](../../releases) から `milky_edge_flow_tools-x.y.z.zip` をダウンロード。
2. Blender で `編集 > プリファレンス > エクステンションを入手 >`（ドロップダウン）`> ディスクからインストール...` を選び、zip を指定。

## 使い方

1. Edit Mode でエッジループを 1 本以上選択（部分選択も可）。
2. 右クリックから `milkyEdgeFlowTools > 交差するフローをリラックス` を選択。
3. Redo パネルで調整:

| オプション | デフォルト | 説明 |
|---|---|---|
| 強度 | 1.0 | 元の位置とリラックス後の位置のブレンド |
| サイドブレンド | 0.0 | リング数が多い側のフロー (0) と少ない側のフロー (1) のブレンド |
| 面角度のしきい値 | 90° | 交差エッジの 2 面の内角がこの値以下の場合、形状を定義するエッジとみなして固定 |
| 剛性 | 1.0 | 再配置の滑らかさ。値が大きいほど固定頂点の影響が遠くまで及ぶ |

## 開発

[uv](https://docs.astral.sh/uv/) が必要です。タスクは `pyproject.toml`（poethepoet）に定義されています。Blender がデフォルトの場所にない場合は環境変数 `BLENDER` で指定してください。

| コマンド | 内容 |
|---|---|
| `uv run poe test` | 全テスト（ユニット + ヘッドレス Blender 統合） |
| `uv run poe test-core` | ユニットテストのみ（Blender 不要） |
| `uv run poe test-blender` | ヘッドレス Blender での統合テスト |
| `uv run poe validate` | Extension マニフェストの検証 |
| `uv run poe build` | 配布 zip を `dist/` にビルド |

完全な仕様は [requirements.md](requirements.md) を参照してください。リリースは release-please で自動化されており、コミットメッセージは [Conventional Commits](https://www.conventionalcommits.org/ja/) に従います。
